About SpotViz

In addition to hopefully providing an interesting service to visualize the geographic location of received signals over time, this app has been (and still is) a personal development playground for trying out some different technologies for building a webapp in a 'single page application' style, using AngularJS for the front-end, and a Java EE7 based back end. The source for the app is available on GitHub here.

Here's a summary of some of the technologies/libraries used to build this app:

The front-end client for this site is using AngularJS. Data for display is retrieved as needed by calling JAX-RS resources running on WildFly. All the processing of the data into intervals for display is achieved with JavasScript and AngularJS.

The backend for the app, mainly JAX-RS resources and data access to a MongoDB database, is running on a WildFly 8.1 cartridge hosted on OpenShift.

Location data for received callsigns is looked up using HamQTH webservice API. See here for more info.
