About SpotViz

This webapp provides the ability to visualize the geographic location of received signals over time by parsing WSJT-X log files. The source for the app is available on GitHub here.

Here's a summary of some of the technologies/libraries used to build this app:

The front-end client for this site is using AngularJS. Data for display is retrieved as needed by calling JAX-RS resources running on WildFly. All the processing of the data into intervals for display is achieved with JavaScript and AngularJS.

The backend for the app, mainly JAX-RS resources and data access to a MongoDB database, is running on WildFly 17.

Location data for received callsigns is looked up using HamQTH webservice API. See here for more info.
